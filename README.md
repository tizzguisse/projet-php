# Mini Projet PHP - Annuaire de Contacts

Application web PHP pour gérer un annuaire de contacts (prénom, nom, email, téléphone, adresse).

## Fonctionnalités
- **Ajouter** un contact
- **Modifier** un contact existant
- **Supprimer** un contact
- Stockage des données en fichier JSON

## Prérequis
- XAMPP (Apache + PHP)
- Navigateur web

## Installation
1. Placer le dossier dans `C:\xampp\htdocs\Projet php`
2. Démarrer Apache dans XAMPP
3. Ouvrir : `http://localhost/Projet%20php/`

## Fichiers
- `index.php` - Application principale
- `contacts.json` - Stockage des données
- `README.md` - Ce fichier
